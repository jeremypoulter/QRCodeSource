QRCodeSource
============

AVISynth plugin to provide a means to embed QR codes in video

Building
========

Visual Studio 2015
==================

Open QRCodeSource.sln and build

Linux
=====

 ./autogen.sh
 ./configure --prefix=$HOME/vapoursynth
 make
 make install
